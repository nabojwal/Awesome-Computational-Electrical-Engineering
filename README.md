# 📘 Awesome Computational-Electrical Engineering

A personally curated collection of foundational textbooks spanning Electrical Engineering, Computer Science, and applied mathematics.

This repository is designed to serve as a structured academic resource for students, researchers, and self-learners building strong theoretical and computational foundations.

---

## 👤 Maintainer

**Nabojwal Acharjee**  
Graduate Student, North Eastern Regional Institute of Science and Technology (NERIST), Arunachal Pradesh  

---

## 🎯 Purpose of This Repository

This collection has been assembled with the following objectives:

- To organize high-quality academic resources in one accessible location  
- To support systematic learning across core engineering disciplines  
- To provide a reference backbone for advanced research in computational sciences  

The selection reflects a curriculum-oriented approach, emphasizing conceptual depth and mathematical rigor.
